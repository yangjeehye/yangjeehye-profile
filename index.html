UX/UI

내가 생각하는 사용자경험 - USER EXPERIENCE

제가 생각하는 UX란, 사용자가 늘 생활하던 경험이 바뀌게 되는 것이라고 생각합니다.
여기서 경험이 바뀌게 될 때 사용자가 편리함을 느껴야 하며 지속적으로
사용하고 싶게 만드는 게 가장 중요하다고 생각이듭니다.
예를 들면 은행 영업점 창구에서 직접 번호표를 뽑고 순서를 기다리던 사용자들이
모바일로 미리 번호표를 예약하고 대기시간 없이 창구를 편하게 이용할 수 있도록 하는
것을 말합니다. 이처럼 사용자가 느낄 수 있는 불편함과 아쉬움 같은 부정적인 감정을
최소화시키는 것이라고 생각합니다.

  • UX-DESIGN
  • UI-DESIGN
  1. UX 디자인은 사용자가 어떤 서비스를
    직·간접적으로 이용하면서 느끼는 반응과 행동들
    같은 경험을 총체적으로 설계하는 것 입니다.
  2. UI 디자인은 사용자가 서비스나
    제품을 어떤 방식으로 이용하도록
    만드는것을 디자인하는 것 입니다.

끊임없이 노력 · 도전 · 열정을 다하는 저의 퍼블리싱 이야기입니다.
사전면접 질문을 선택하시면 정리된 답변을 보실 수 있습니다.

1. HTML5의 가장 큰 특징은 무엇일까요?

웹 브라우저마다 HTML을 해석하는 방식의 차이로 인한 오류와 혼란을 피하기 위하여
구현방식을 단순화 하였다는 점과 <header>,<nav>,<footer>와 같은 시멘틱태그를 제공하여
웹 데이터의 짜임을 의미있게 구현할 수 있다는 점이 있습니다.

2. 웹 접근성(Web Accessibility)은 어떻게 설명할 수 있을까요?

어떠한 사용자(장애인,고령자 등), 어떠한 기술환경에서도 사용자가 전문적인 능력 없이
웹사이트에서 제공하는 모든 정보에 접근할 수 있도록 보장하는 것입니다.
이처럼 대상이 한정되지 않고 모두가 사용하기 편하도록하는 사용자에게 context를 제공하는
스크린 리더나 글꼴을 확대하도록 허용하는 등 이 모든게 웹 접근성입니다.

3. 자바스크립트 라이브러리(Library) 사용시 가장 큰 장점은 무엇일까요?

사람이 이해하기 쉽고 간결한 코드를 구사하여 만들어 내고자 하는 것을
보다 빨리 생산할 수 있다는 점이 가장 큰 장점입니다.

4. 시간반복함수는 무엇이고 이를 정지시키는 함수는 무엇인지 아시나요?

시간반복함수는 'setInterval()' 메소드로 기준 간격을 두어 주기적으로
이벤트를 발생 시키고 싶을 때 사용하는 것이고, 이를 정지시키는 함수는 'clearInterval()' 메소드 입니다.

5. CSS 미디어 쿼리는 어떤 역할을 수행하는지 알려드립니다.

미디어 쿼리(mediaqueri)는 사이트에 접속하는 기기에 따라 특정한 CSS 스타일을 사용하도록 도와주는 소프트웨어 모듈입니다.
min이나 max등 앞에 접두어에 따라 미디어 쿼리를 넣는 순서를 유의하여 제작하면
웹 사이트에 접속하는 기기에 따라서 레이아웃이 바뀌는 반응형으로 제작할 수 있습니다.

더욱 궁금하신 점은 면접에서 말씀드리겠습니다. 감사합니다.
